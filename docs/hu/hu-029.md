# HU-029: Permisos de Módulos por Empresa

**Epic:** EP-006 - Sistema de Permisos de Módulos

## Descripción

Como super-admin de la plataforma, quiero activar o desactivar módulos específicos para cada empresa, controlando qué funcionalidades tiene disponibles cada cliente.

## Criterios de Aceptación

- [x] El super-admin puede ver los módulos activos de una empresa
- [x] El super-admin puede activar un módulo para una empresa
- [x] El super-admin puede desactivar un módulo para una empresa
- [x] Al crear una empresa, por defecto todos los módulos están activos
- [x] Al desactivar un módulo, automáticamente se desactivan todos sus submódulos
- [x] La empresa no puede acceder a un módulo desactivado

## API Endpoints

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/api/platform/empresas/{id}/permisos` | Ver permisos de módulos y submódulos |
| PUT | `/api/platform/empresas/{id}/permisos` | Actualizar permisos |

## Estructura del Request

```json
{
  "modulos": [
    {
      "moduloId": 1,
      "activo": true
    }
  ]
}
```

## Reglas de Negocio

1. Solo el super-admin (PLATFORM_ADMIN) puede modificar permisos
2. Al desactivar un módulo padre, los submódulos también se desactivan implícitamente
3. La empresa no puede acceder a endpoints de módulos desactivados

## Database Modifications

### Tabla: permiso_empresa
```sql
CREATE TABLE IF NOT EXISTS permiso_empresa (
    id BIGSERIAL PRIMARY KEY,
    empresa_id BIGINT NOT NULL REFERENCES empresa(id),
    modulo_id BIGINT NOT NULL REFERENCES modulo(id),
    submodulo_id BIGINT REFERENCES submodulo(id),
    activo BOOLEAN NOT NULL DEFAULT true,
    created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP,
    deleted_at TIMESTAMP,
    UNIQUE(empresa_id, modulo_id, submodulo_id)
);

-- Índices
CREATE INDEX idx_permiso_empresa_empresa ON permiso_empresa(empresa_id);
CREATE INDEX idx_permiso_empresa_modulo ON permiso_empresa(modulo_id);
CREATE INDEX idx_permiso_empresa_submodulo ON permiso_empresa(submodulo_id);
```

### Datos Iniciales (al crear empresa)

```sql
-- Al crear una empresa, insertar todos los módulos activos por defecto
INSERT INTO permiso_empresa (empresa_id, modulo_id, submodulo_id, activo)
SELECT 
    :empresa_id,
    m.id,
    NULL,
    m.activo
FROM modulo m
WHERE m.activo = true;

-- Insertar permisos de submódulos activos
INSERT INTO permiso_empresa (empresa_id, modulo_id, submodulo_id, activo)
SELECT 
    :empresa_id,
    s.modulo_id,
    s.id,
    s.activo
FROM submodulo s
WHERE s.activo = true;
```
