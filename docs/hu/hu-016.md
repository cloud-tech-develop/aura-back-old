# HU-016: Registro de Logs de Facturación

## Descripción
Sistema de auditoría y trazabilidad para facturas electrónicas. Cada acción relevante sobre una factura debe generar un registro en el log para mantener historial completo de cambios, estados y eventos.

## Criterios de Aceptación

### Eventos a Registrar
- [x] **CREACION**: Cuando se crea una factura automáticamente desde una venta
- [x] **CAMBIO_ESTADO_DIAN**: Cuando cambia el estado de validación DIAN
- [x] **ENVIO_DIAN**: Cuando se envía la factura a la DIAN
- [x] **RESPuesta_DIAN**: Cuando se recibe respuesta de la DIAN
- [x] **ANULACION**: Cuando se anula una factura
- [x] **ACTUALIZACION**: Cuando se actualizan datos de la factura

### Estructura del Log
- [x] `factura_id`: ID de la factura relacionada
- [x] `evento`: Tipo de evento (CREACION, CAMBIO_ESTADO_DIAN, etc.)
- [x] `estado_anterior`: Estado anterior (si aplica)
- [x] `estado_nuevo`: Estado nuevo (si aplica)
- [x] `datos`: JSON con datos relevantes del evento
- [x] `usuario_id`: Usuario que ejecutó la acción
- [x] `mensaje`: Descripción legible del evento
- [x] `metadata`: JSON con información adicional
- [x] `created_at`: Fecha y hora del evento

### Implementación
- [x] Servicio `FacturaLogService` con método `registrarLogAsync()`
- [x] Transacción separada (`REQUIRES_NEW`) para no afectar la operación principal
- [x] Tabla `factura_log` con soft delete

### Casos de Uso
1. **Venta a crédito**: Se crea factura + log de CREACION
2. **Envío a DIAN**: Log de ENVIO_DIAN con datos del request
3. **Respuesta DIAN**: Log de RESPUESTA_DIAN con resultado
4. **Anulación**: Log de ANULACION con理由

## Tabla factura_log
```sql
CREATE TABLE IF NOT EXISTS factura_log (
    id BIGSERIAL PRIMARY KEY,
    factura_id BIGINT NOT NULL REFERENCES factura(id),
    evento VARCHAR(50) NOT NULL,
    estado_anterior VARCHAR(50),
    estado_nuevo VARCHAR(50),
    datos JSONB,
    metadata JSONB,
    mensaje TEXT,
    usuario_id BIGINT REFERENCES usuario(id),
    created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP
);

CREATE INDEX idx_factura_log_factura ON factura_log(factura_id);
CREATE INDEX idx_factura_log_created_at ON factura_log(created_at DESC);
```
