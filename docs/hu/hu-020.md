---

## 👤 Historia de Usuario

**Como** usuario del sistema (gerente/administrador)  
**Quiero** visualizar un listado de empleados filtrados por tipo de empleado  
**Para** poder analizar y gestionar la fuerza laboral por categoría

---

## 🧠 Descripción Funcional

El sistema debe mostrar una lista paginada de empleados filtrada por el tipo de empleado seleccionado. El usuario puede seleccionar un tipo de empleado de un dropdown y el sistema mostrará todos los empleados de ese tipo.

---

## ✅ Criterios de Aceptación

### Escenario 1: Listar empleados por tipo
- Dado que el usuario accede a la pantalla de empleados por tipo
- Cuando selecciona un tipo de empleado del dropdown
- Entonces el sistema muestra una lista paginada de empleados de ese tipo

### Escenario 2: Ver todos los tipos en dropdown
- Dado que el usuario está en la pantalla de empleados por tipo
- Cuando el dropdown de tipos de empleado carga
-Entonces el sistema muestra todos los tipos de empleado activos de la empresa

### Escenario 3: Cambiar tipo de empleado
- Dado que el usuario está viendo la lista de empleados de un tipo
- Cuando selecciona un diferente tipo de empleado
-Entonces el sistema actualiza la lista con los empleados del nuevo tipo

### Escenario 4: Ver detalles del empleado
- Dado que el usuario está en la lista de empleados por tipo
- Cuando hace clic en un empleado
-Entonces el sistema muestra los detalles completos del empleado

---

## ❌ Casos de Error

- Si no hay empleados del tipo seleccionado, mostrar mensaje "No hay empleados de este tipo"
- Si el tipo no existe, mostrar error

---

## 🔐 Reglas de Negocio

- Solo mostrar tipos de empleado activos
- Filtrar por empresa_id del JWT
- Incluir paginación

---

## 🎨 Consideraciones UI/UX

- Dropdown con filtro para buscar tipos
- Tabla con paginación
- Mostrar nombre del tipo en la columna correspondiente

---

## 📡 Requisitos Técnicos

- Endpoint GET /api/empleados?tipoEmpleadoId={id} - Filtrar por tipo
- El dropdown de tipos usa GET /api/tipos-empleado

---

## 📎 Dependencias

- HU-018: Gestión de Tipos de Empleado
- HU-019: Asociación de Tipo de Empleado a Empleados

---

## 🚫 Fuera de Alcance

- No incluye edición de empleado desde esta pantalla
- No incluye reporte/exportación
