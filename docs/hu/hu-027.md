# HU-027: Gestión de Módulos

**Epic:** EP-006 - Sistema de Permisos de Módulos

## Descripción

Como super-admin de la plataforma, quiero gestionar (crear, editar, eliminar) los módulos del sistema para definir la estructura de permisos disponible.

## Criterios de Aceptación

- [x] El super-admin puede listar todos los módulos existentes
- [x] El super-admin puede crear un nuevo módulo con nombre, código único, descripción y orden
- [x] El super-admin puede editar un módulo existente
- [x] El super-admin puede eliminar un módulo (si no tiene submódulos asociados)
- [x] No puede haber dos módulos con el mismo código
- [x] Los módulos tienen un campo orden para顯示順序
- [x] Los módulos tienen un campo activo para habilitarlos/deshabilitarlos

## Datos del Formulario

- **Nombre:** texto (obligatorio, max 100 caracteres)
- **Código:** texto único (obligatorio, max 50 caracteres, solo minúsculas y guiones)
- **Descripción:** texto (opcional, max 500 caracteres)
- **Orden:** número entero (opcional, por defecto 0)
- **Activo:** boolean (por defecto true)

## API Endpoints

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/api/platform/modulos` | Listar módulos |
| POST | `/api/platform/modulos` | Crear módulo |
| PUT | `/api/platform/modulos/{id}` | Actualizar módulo |
| DELETE | `/api/platform/modulos/{id}` | Eliminar módulo |

## Reglas de Negocio

1. El código debe ser único en toda la plataforma
2. Al eliminar un módulo, deben eliminarse en cascada sus submódulos y permisos asociados
3. Solo el super-admin (PLATFORM_ADMIN) puede acceder a estos endpoints

## Database Modifications

### Tabla: modulo
```sql
CREATE TABLE IF NOT EXISTS modulo (
    id BIGSERIAL PRIMARY KEY,
    nombre VARCHAR(100) NOT NULL,
    codigo VARCHAR(50) NOT NULL UNIQUE,
    descripcion VARCHAR(500),
    orden INTEGER NOT NULL DEFAULT 0,
    activo BOOLEAN NOT NULL DEFAULT true,
    created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP,
    deleted_at TIMESTAMP
);

-- Índices
CREATE INDEX idx_modulo_codigo ON modulo(codigo);
CREATE INDEX idx_modulo_activo ON modulo(activo);
CREATE INDEX idx_modulo_orden ON modulo(orden);
```
