# HU-028: Gestión de Submódulos

**Epic:** EP-006 - Sistema de Permisos de Módulos

## Descripción

Como super-admin de la plataforma, quiero gestionar (crear, editar, eliminar) los submódulos del sistema para definir la estructura de permisos granular.

## Criterios de Aceptación

- [x] El super-admin puede listar todos los submódulos existentes
- [x] El super-admin puede crear un nuevo submódulo asociado a un módulo padre
- [x] El super-admin puede editar un submódulo existente
- [x] El super-admin puede eliminar un submódulo
- [x] No puede haber dos submódulos con el mismo código dentro del mismo módulo
- [x] Los submódulos tienen un campo orden para顯示順序
- [x] Los submódulos tienen un campo activo para habilitarlos/deshabilitarlos

## Datos del Formulario

- **Módulo Padre:** selector (obligatorio)
- **Nombre:** texto (obligatorio, max 100 caracteres)
- **Código:** texto único por módulo (obligatorio, max 50 caracteres, solo minúsculas y guiones)
- **Descripción:** texto (opcional, max 500 caracteres)
- **Orden:** número entero (opcional, por defecto 0)
- **Activo:** boolean (por defecto true)

## API Endpoints

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/api/platform/submodulos` | Listar submódulos |
| GET | `/api/platform/submodulos/modulo/{moduloId}` | Listar submódulos por módulo |
| POST | `/api/platform/submodulos` | Crear submódulo |
| PUT | `/api/platform/submodulos/{id}` | Actualizar submódulo |
| DELETE | `/api/platform/submodulos/{id}` | Eliminar submódulo |

## Reglas de Negocio

1. El código debe ser único dentro del módulo padre
2. Al eliminar un submódulo, deben eliminarse sus permisos asociados
3. Solo el super-admin (PLATFORM_ADMIN) puede acceder a estos endpoints

## Database Modifications

### Tabla: submodulo
```sql
CREATE TABLE IF NOT EXISTS submodulo (
    id BIGSERIAL PRIMARY KEY,
    modulo_id BIGINT NOT NULL REFERENCES modulo(id),
    nombre VARCHAR(100) NOT NULL,
    codigo VARCHAR(50) NOT NULL,
    descripcion VARCHAR(500),
    orden INTEGER NOT NULL DEFAULT 0,
    activo BOOLEAN NOT NULL DEFAULT true,
    created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP,
    deleted_at TIMESTAMP,
    UNIQUE(modulo_id, codigo)
);

-- Índices
CREATE INDEX idx_submodulo_modulo ON submodulo(modulo_id);
CREATE INDEX idx_submodulo_codigo ON submodulo(codigo);
CREATE INDEX idx_submodulo_activo ON submodulo(activo);
```
